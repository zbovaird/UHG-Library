# Interactive conceptual demo

Static HTML demo (`index.html`) for building intuition about projective UHG (open locally in a browser).

This is **educational material**, not part of the `uhg` package install. Optional dependencies for richer demos may be listed here in the future.

For the full library API, see the [main README](../../README.md) and [docs/reference/public-api.md](../../docs/reference/public-api.md).
